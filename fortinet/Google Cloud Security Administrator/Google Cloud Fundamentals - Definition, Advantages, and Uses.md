### Google Cloud

Google Cloud is defined as a public cloud vendor that provides infrastructure and services, such as Compute Engine, over the public internet .These resources are accessible from anywhere and anytime as needed.

The slide outlines the primary advantages that Google Cloud offers:

- Cost based on use (low cost of ownership).
- Automated deployments.
- Scalability.
- Security.
- Recovery.
- Reliability.

Google Cloud is well-suited for several specific deployment scenarios 2:

- Data storage and data archival.
- Application hosting.
- Hosting for latency-intolerant or mission-critical web tiers.
- On-demand hosting for microsites and applications.
    
Providing an auto-scaling environment for large applications.
