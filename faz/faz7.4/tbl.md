# Replace the FortiAnalyzer device

When you need to move logs to a new FortiAnalyzer device, use one of the following methods:

- Use log forwarding in aggregation mode. See _Log Forwarding_ in the _FortiAnalyzer Administration Guide_.
- Use log fetching (Fetcher Management). See _Fetcher Management_ in the _FortiAnalyzer Administration Guide_.
